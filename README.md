# question 6:
# Un conflit Git survient lorsqu’il y a des modifications concurrentes incompatibles sur les mêmes lignes ou sections de code dans différents commits ou branches. Git ne peut pas automatiquement résoudre ces conflits et vous devez intervenir manuellement pour décider quelle version du code conserver. Les conflits sont fréquents lors de l'intégration de branches ou lors de la mise à jour du code local avec des changements provenant du dépôt distant.
